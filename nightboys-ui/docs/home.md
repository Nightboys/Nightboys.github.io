    npm i vant jquery webpack-zepto -S

    npm i vant@next -S

### 特性

- 新增目录树组件
- 新增日历组件，区别于 vant 自身日历组件
- 新增表格分页及滚动组件，适用于移动端表格展示
- 新增顶部筛选组件，用于筛选大容量数据
- 新增省市区多级联动组件
